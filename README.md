# Conversor de Medidas

Converta temperatura, peso, distância e moeda direto pelo terminal.

## Como usar

```bash
python coversor.py <valor> <unidade de origem> <unidade de destino>
```

## Exemplos

```bash
python coversor.py 100 usd brl
python coversor.py 37 c f
python coversor.py 70 kg lb
python coversor.py 10 km mi
```

## Unidades suportadas

| Categoria   | Unidades                                 |
|-------------|------------------------------------------|
| Temperatura | c, f, k                                  |
| Peso        | kg, g, lb, oz, t                         |
| Distância   | m, km, cm, mm, mi, ft, in, yd            |
| Moeda       | usd, brl, eur e mais de 150 moedas       |

## Requisitos

- Python 3.6+
